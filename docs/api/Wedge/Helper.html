find(loadpath, relname, options={}) click to toggle source

Given a loadpath, a file’s relative path, relname, and options hash, determine a matching file exists. Unless :load option is true, this will check for each viable Ruby suffix. If a match is found the full path to the file is returned, otherwise nil.
